Model Overview

The SVM model was trained with four kernels: linear, polynomial, RBF, and sigmoid. The linear kernel performed best and was selected for final training.

Why SVM?

Effective for classification tasks, SVMs handle non-linear data and excel in high-dimensional spaces, making them well-suited for dropout prediction.

Performance

The linear kernel achieved 80.9% accuracy on the test set, outperforming the other kernels.

Model Overview

We trained the Random Forest model in two stages: first with default parameters, then with hyperparameter tuning using Grid Search for optimal performance.

Why Random Forest?

Random Forest is an ensemble method that improves prediction accuracy by combining multiple decision trees. It reduces overfitting and captures complex data patterns.

Performance Insights

With default parameters, the model achieved 99.9% accuracy on training data, 85.5% on validation. After tuning, the validation accuracy improved slightly to 85.0%, but test set accuracy was 82.8%, indicating overfitting.

Model Overview

The KNN model was trained using Grid Search for optimal parameters and the elbow method to determine the best K value.

Why K-Nearest Neighbors?

KNN is a simple, effective classification algorithm that predicts based on the majority class of nearest neighbors. It works well for complex, non-linear data distributions.

Performance Insights

After applying Grid Search, the model achieved 83.9% accuracy on training data and 82.3% on validation. The elbow method suggested an optimal K value of 11, balancing accuracy between training and validation sets.
